# Fake News Detection using NLP + LSTM

## Project Overview
This project detects fake news using deep learning (LSTM) and NLP techniques with GloVe embeddings.

## Tech Stack
- Python
- TensorFlow / Keras
- NLP (Tokenization, Embeddings)
- LSTM Neural Network

## Dataset
Kaggle Fake and Real News dataset (not included due to size)

## Model
- LSTM with GloVe embeddings
- Binary classification (Fake / Real)

## Result
Achieves high accuracy (~95–99%) depending on training.

## Note
Large files (dataset and embeddings) are excluded from repository due to GitHub size limits.
